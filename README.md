[inicializacion]:
1.- Ejecutar el archivo "Inicializador.bat"(esto solo para windows)
2.- abrir la carpeta "backend"
3.- ejecutar el entorno virtual (.venv\Scripts\activate)
4.- instalar las librerias con 
pip install -r "./requirements.txt"
5.-iniciar el servidor con 
python main.py

[Version]:
version Alpha (sujeto a cambios en actualizaciones)
